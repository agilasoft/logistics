# Sustainability Workspace - Reports & Card Groups Guide

## Reports to Add in Links Section

Add these reports organized into the following card groups:

### 1. **Core Metrics Reports** (Card Break)
This group contains the fundamental sustainability measurement reports:
- Sustainability Metrics
- Sustainability Metrics Report
- Sustainability Dashboard Report

### 2. **Environmental Impact Reports** (Card Break)
Reports focused on environmental measurements:
- Carbon Footprint Report
- Energy Consumption Report

### 3. **Goals & Compliance Reports** (Card Break)
Reports for tracking goals and compliance:
- Sustainability Goals Report
- Sustainability Compliance Report

### 4. **Analysis Reports** (Card Break)
Advanced analysis and cross-module reports:
- Cross Module Summary Report
- Trend Analysis Report

## How to Add in UI:

### Step 1: Add Card Breaks (Section Headers)
For each group above, add a **Card Break** link first:

1. **Card Break 1: Core Metrics Reports**
   - Type: `Card Break`
   - Link Type: `DocType` (or leave empty)
   - Label: `Core Metrics Reports`
   - Link Count: `3` (number of reports in this group)

2. **Card Break 2: Environmental Impact Reports**
   - Type: `Card Break`
   - Label: `Environmental Impact Reports`
   - Link Count: `2`
