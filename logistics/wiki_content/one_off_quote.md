# One Off Quote

**One Off Quote** is a quick pricing document for ad-hoc or one-time freight requests. It supports Air Freight, Sea Freight, Transport, and Warehousing services with simplified pricing.

Unlike Sales Quote, One Off Quote is designed for quick turnaround quotes without full routing and multi-leg setup. It can be converted to the appropriate booking or order.

To access One Off Quote, go to:

**Home > Pricing Center > One Off Quote**

## 1. Prerequisites

Before creating a One Off Quote, it is advised to set up the following:

- Customer (from ERPNext)
- Port/Airport masters (for freight)
- [Transport Zone](welcome/transport-zone) (for transport)

## 2. How to Create a One Off Quote

1. Go to the One Off Quote list, click **New**.
2. Enter **Quote Date** and select **Customer**.
3. Select **Product Type** (Air, Sea, Transport, Warehouse).
4. Add origin, destination, and cargo details.
5. Add **Charges** (freight, fuel, handling, etc.).
6. **Save** the document.

## 3. Features

### 3.1 Creating Documents from One Off Quote

From a submitted One Off Quote, you can create the corresponding Air Booking, Sea Booking, Transport Order, or Warehouse Job.
