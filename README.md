# CargoNext - Comprehensive Logistics Management System

CargoNext is a comprehensive logistics management system built on Frappe Framework.

## Features

- Transport Management
- Vehicle Tracking
- Route Optimization
- Transport Leg Management
- Run Sheet Management
- And more...

## Installation

This app is part of a Frappe bench. Install it using:

```bash
bench get-app logistics
bench install-app logistics
```
