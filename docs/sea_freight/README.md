## Key Concepts

### Sea Shipment
The main transaction document representing a single sea freight shipment from origin to destination.

### Master Bill
Master Bill of Lading issued by shipping lines for consolidated cargo.

### Consolidation
Process of combining multiple shipments into a single consolidation for transportation.

### Master Data
Foundation data including shipping lines, ports, container yards, and terminal operators.

## Module Integration

The Sea Freight module integrates with:

- **Air Freight**: For multi-modal shipments
- **Transport**: For inland transportation
- **Customs**: For customs clearance
- **Warehousing**: For container yard operations
- **Pricing Center**: For rate calculation
