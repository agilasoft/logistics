## Key Concepts

### Air Shipment
The main transaction document representing a single air freight shipment from origin to destination.

### Master Air Waybill
Master Air Waybill issued by airlines for consolidated cargo.

### Consolidation
Process of combining multiple shipments into a single consolidation for transportation.

### Master Data
Foundation data including airlines, airports, flight schedules, and ULDs.

## Module Integration

The Air Freight module integrates with:

- **Sea Freight**: For multi-modal shipments
- **Transport**: For inland transportation
- **Customs**: For customs clearance
- **Warehousing**: For cargo terminal operations
- **Pricing Center**: For rate calculation
- **Sustainability**: For carbon footprint tracking
